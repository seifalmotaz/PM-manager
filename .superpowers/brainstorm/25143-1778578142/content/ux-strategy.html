Fast & Reliable UX Strategy

Focusing on "speed at every touch" to match the Todoist experience.

1

Universal Command Palette (CMD+K)

Instant navigation and task creation. Type "/" for actions, "@" for people, "#" for projects. Never take your hands off the keyboard.

2

Optimistic UI & Skeleton States

Actions (moving tasks, changing priority) happen instantly on screen before the server confirms. "Zero latency" perception.

3

Global Keyboard Shortcuts

"Q" for quick-add, "S" for search, "1-4" for sidebar navigation. Familiar patterns for Todoist/Linear power users.

4

Contextual Power-Actions

Keep cards clean. Reveal "Move", "Assign", "Due Date" on hover or right-click. Maintains Todoist simplicity with Jira power.

Performance Mandate

We'll target sub-100ms response times for all UI transitions. Using Svelte 5's runes for fine-grained reactivity and minimal re-renders.
